# Team K9 Boxing — v1

First version of a professional website built for boxing coach Cornelius "K9" Bundrage. 
This was my first real client project, built with vanilla HTML, CSS, JavaScript, and PHP 
and deployed with a custom domain and SSL certificate through IONOS.

**Live Site:** [teamk9boxing.com/K9Boxing.html](https://teamk9boxing.com/K9Boxing.html)

## Features

- Photo gallery
- Services section
- PHP contact form for client inquiries
- Custom domain with SSL via IONOS

## Tech Stack

- HTML, CSS, JavaScript — frontend
- PHP — contact form / backend
- IONOS — hosting, domain, SSL
